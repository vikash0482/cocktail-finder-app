# 🍹 Cocktail Finder App

A full-stack web application that allows users to search for cocktails and view their details.

## 🚀 Features
- Search any cocktail by name
- Displays exact match results
- Clean UI using EJS templating
- Fetches real-time data from CocktailDB API

## 🛠️ Tech Stack
- Node.js
- Express.js
- EJS
- Axios
- CSS

## 📂 Project Structure
- index.js → Backend logic
- views/ → EJS templates
- public/ → CSS files

## 📦 Installation

bash....>
* npm install
* npm start 


API Used;
https://www.thecocktaildb.com/api.php
